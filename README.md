## Gira

## Setup Instructions

**Prerequisites**
- Git
- Node.js
- Python 3.11+
- Docker Desktop

**Check versions**
- node -v
- npm -v
- python --version
- docker --version

**Download the project from Github**

**Start database** <br>
`docker compose up -d` <br>
`docker ps` <br>

**Backend setup** <br>
`cd backend` <br>
`python -m venv gira` <br>
`gira\Scripts\activate` <br>

**Install dependencies** <br>
`pip install -r requirements.txt` <br>
